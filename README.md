libffi_binaries
=====

Compiled binary files of the "libffi" library for the various platforms.

Version: 0.6.6

**Supported platforms**

- ARM Android
- ARM Linux
- X86 Linux
- X86 Windows
- X86_64 Linux
- X86_64 Mac OS
- X86_64 Windows

Special thanks to [Kenneth Endfinger](https://github.com/kaendfinger) for compiling the binaries:

- ARM Android
- ARM Linux
- X86_64 Mac OS X
